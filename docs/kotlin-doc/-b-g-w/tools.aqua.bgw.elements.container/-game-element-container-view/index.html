GameElementContainerView

sealed class GameElementContainerView<T : GameElementView> : DynamicView
Content copied to clipboard

GameElementContainerView is the abstract super class for containers that can contain ElementViews or its subclasses. It provides the ObservableList to store ElementViews and some useful methods to work on said list.

Parameters

Height for this gameElementContainerView. Default: 0.

Width for this gameElementContainerView. Default: 0.

Horizontal coordinate for this GameElementView. Default: 0.

posY

Vertical coordinate for this GameElementView. Default: 0.
