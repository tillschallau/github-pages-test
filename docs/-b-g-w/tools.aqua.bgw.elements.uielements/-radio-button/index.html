RadioButton

open class RadioButton(height: Number, width: Number, posX: Number, posY: Number, text: String, fontSize: Number, isSelected: Boolean, toggleGroup: ToggleGroup?) : ToggleButton
Content copied to clipboard

A RadioButton may be used as a Button that is either selected or not selected. An important feature of ToggleButtons is the ToggleGroup. A RadioButton has the same functionality as a ToggleButton but a different visual representation.

See also

tools.aqua.bgw.elements.uielements.ToggleButton
tools.aqua.bgw.elements.uielements.ToggleGroup

Height for this ToggleButton. Default: 0.

Horizontal coordinate for this ToggleButton. Default: 0.

posY

Vertical coordinate for this ToggleButton. Default: 0.

Label for this ToggleButton. Default: empty string.

isSelected

the initial state for this ToggleButton. Default: false.

the ToggleGroup of this ToggleButton. Default: null.
